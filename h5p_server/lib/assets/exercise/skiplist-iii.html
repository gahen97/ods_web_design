Given a Skiplist, update the Skiplist as the following operations are performed.
  • SET OPERATION:
    1. Drag your mouse over an element to add it to the search path.
    2. Complete the search path for the element to set its new value.
  • REMOVE OPERATION:
    1. Drag your mouse over an element to add it to the search path.
    2. While moving through the list, update the pointers if needed to remove the element.
    3. Check your answer by clicking on an element.
  • ADD OPERATION:
    1. Drag your mouse over an element to add it to the search path.
    2. While moving through the path, the new element may be added to the list.
    3. If a new element is added, update the pointers to add the element.
