Starting with an Array Stack that is initially empty, illustrate the evolution of the stack as the following operations are performed.
For this exercise, array size is assumed to be:
  • Doubled when array becomes full
  • Halved when array becomes less than one third full
  • Drag elements to move them around in the stack.
  • Drop the trash can on any element to delete it.
  • Drag the end of the stack to resize it.
  • Press the Check button to check your answer.
